## spring-fom

一个轻量级的任务管理，主要针对定时任务，相对Spring Schedule提供了更多的场景支持

- 依赖

```xml
<dependency>
    <groupId>com.cowave.commons</groupId>
    <artifactId>spring-fom</artifactId>
    <version>2.7.1</version>
</dependency>
```
